# object-encoder <a href="https://www.npmjs.com/package/object-encoder"><img src="https://img.shields.io/npm/v/object-encoder.svg" /></a>

## 介绍

对象编码器（Object Encoder）支持将类实例编码为包含还原源类型所需字段信息的普通 JS 对象。
